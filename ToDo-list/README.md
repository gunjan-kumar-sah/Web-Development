# ✅ Todo Iist (JavaScript Project)

A simple and interactive Todo List application built using **HTML, CSS, and JavaScript**.  
This project helps users manage daily tasks efficiently by adding and deleting tasks dynamically.

---

# 🚀 Features

✔ Add new tasks  
✔ Delete tasks  
✔ Real-time UI updates  
✔ Simple and clean interface  
✔ Beginner-friendly logic  

---

# 🛠️ Technologies Used

- HTML5 (Structure)
- CSS3 (Styling)
- JavaScript (Functionality)

---

# ⚙️ How It Works

1. User enters a task in input field  
2. Clicks “Add” button  
3. Task is stored in an array  
4. UI updates dynamically  
5. Tasks can be deleted using delete button  

---

# 🧠 Key Concepts Used

✔ DOM Manipulation  
✔ Arrays in JavaScript  
✔ Event Handling  
✔ Dynamic HTML rendering  

---

# 🎯 Learning Outcome

This project improves understanding of:
- JavaScript basics
- UI updates without page reload
- Array handling
- User interaction logic
