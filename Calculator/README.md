# 🧮 Calculator App (JavaScript Project)

A simple and responsive calculator built using **HTML, CSS, and JavaScript**.  
It performs basic arithmetic operations with a clean UI.

---

# 🚀 Features

✔ Addition, Subtraction, Multiplication, Division  
✔ Real-time calculation  
✔ Clean button interface  
✔ Responsive design  
✔ Simple user experience  

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

# ⚙️ How It Works

1. User clicks buttons (numbers/operators)  
2. Input is displayed on screen  
3. Expression is evaluated using JavaScript  
4. Result is shown instantly  

---

# 🧠 Key Concepts Used

✔ Event Listeners  
✔ DOM Manipulation  
✔ JavaScript Operators  
✔ eval() function (basic calculation logic)  

---

This project helps in understanding:

- Mathematical operations handling
- Frontend structuring
