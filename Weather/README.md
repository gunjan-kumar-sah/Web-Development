# 🌦️ Weather App (JavaScript | HTML | CSS)

A modern, responsive and real-time Weather Application built using **HTML, CSS, and JavaScript**.  
It uses the **OpenWeather API** to fetch live weather data for any city across the world.

This project is built as a **portfolio-ready frontend project** to demonstrate real-world API integration, UI/UX design, and JavaScript logic skills.

---

# 🚀 Live Features

✔ Search weather by any city name (Worldwide)  
✔ Real-time temperature in Celsius (°C)  
✔ Weather condition (Clear, Clouds, Rain, Snow etc.)  
✔ Humidity percentage display  
✔ Wind speed display  
✔ Dynamic weather icons based on condition  
✔ Loading state for better user experience  
✔ Error handling for invalid city names  
✔ Responsive and clean UI design  
✔ Smooth user interaction  

---

# 🧠 How This Project Works (Real Logic Flow)

1. User enters a city name in input field  
2. JavaScript captures input using DOM manipulation  
3. API request is sent to OpenWeather server  
4. Server responds with JSON weather data  
5. Data is extracted and displayed dynamically  
6. UI updates instantly without page reload

# ❗ Problems Faced During Development & Solutions

### 1. API Key Issue
Problem:
- API key was not working or showing "Invalid key"

Solution:
- Correctly activated API key from OpenWeather dashboard
- Properly placed in JavaScript file without extra spaces

---

### 2. City Not Found Error
Problem:
- Invalid or misspelled city names caused errors

Solution:
- Implemented proper error handling using API response code check (`data.cod !== 200`)

---

### 3. UI Not Updating Properly
Problem:
- Incorrect DOM selectors or missing class references

Solution:
- Fixed class names and ensured proper display switching (`none / flex`)

---

### 4. Data Not Displaying
Problem:
- Improper JSON parsing or wrong API response handling

Solution:
- Added proper async/await flow and response validation
