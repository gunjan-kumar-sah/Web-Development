# 🧠 Quiz App (JavaScript Project)

An interactive Quiz Application built using **HTML, CSS, and JavaScript**.  
It allows users to answer multiple-choice questions and see their final score.

---

# 🚀 Features

✔ Multiple-choice questions  
✔ Score calculation  
✔ Instant feedback  
✔ Next question navigation  
✔ Final result display  
✔ Simple UI design  

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

# ⚙️ How It Works

1. Question is displayed on screen  
2. User selects an answer  
3. JavaScript checks correctness  
4. Score is updated  
5. Next question is shown  
6. Final score is displayed at end
