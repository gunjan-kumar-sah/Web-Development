# 🧠 How the Game Works

1. Player O starts the game  
2. Players take turns clicking boxes  
3. Each move updates the game state  
4. System checks for winner after every move  
5. If all boxes are filled without winner → Draw  
6. Game can be reset anytime  

---

# 🏆 Win Logic

The game checks these winning combinations:

- Rows: (0,1,2), (3,4,5), (6,7,8)  
- Columns: (0,3,6), (1,4,7), (2,5,8)  
- Diagonals: (0,4,8), (2,4,6)  

If any combination matches, the player is declared winner.
